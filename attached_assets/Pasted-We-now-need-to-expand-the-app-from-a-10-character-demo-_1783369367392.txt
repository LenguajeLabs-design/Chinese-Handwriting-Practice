Build the following features:

1. HSK Level Selector
- User can choose HSK 1, 2, 3, 4, 5, or 6.
- Show progress for each level.

2. Deck Builder
- Automatically split each HSK level into decks of 10 vocabulary items.
- Name them:
  HSK 1 Deck 1
  HSK 1 Deck 2
  HSK 1 Deck 3
- Keep the deck size configurable.

3. Today’s Practice
- Generate a daily practice set of 10 items.
- Use this mix:
  7 new items from the current deck
  3 review items from difficult or previously attempted characters
- If there are not enough review items, fill with new items.

4. Progress Tracking
Use localStorage to save:
- currentLevel
- currentDeckByLevel
- completedDecks
- attemptedCharacters
- difficultCharacters
- completedCharacters
- dailyPracticeHistory
- lastPracticeDate

5. Completion Flow
When the user completes a deck:
- show a friendly completion screen
- unlock the next deck
- offer three choices:
  Continue to next deck
  Review difficult characters
  Practice from memory

6. Character Practice From Words
Because Hanzi Writer practices one character at a time:
- If the item is one character, practice that character.
- If the item is a word with multiple characters, practice each character in sequence.
- Show:
  Word: 中国
  Character 1 of 2: 中
  Character 2 of 2: 国

- DeckCompletion component
- ProgressPanel component
